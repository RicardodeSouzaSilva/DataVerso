# 📊 Projetos Disponíveis

## 🧠 Desbalanceamento de Classes

Primeiro estudo desta seção.

Neste projeto explorei um dos desafios mais importantes em Machine Learning supervisionado: o desbalanceamento de classes.

O estudo demonstra como distribuições desiguais podem impactar diretamente a performance preditiva dos modelos e gerar métricas enganosas, especialmente quando analisamos apenas a acurácia.

</p>

### 🔍 O que foi explorado

- Conceito de classes desbalanceadas
- Impacto em modelos preditivos
- Problema da alta acurácia enganosa
- Técnicas de balanceamento
- Comparação entre abordagens
- Avaliação correta de métricas
- Interpretação dos resultados
